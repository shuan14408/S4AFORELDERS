自己動手做：接線手冊 🛠️

照著圖一條線一條線接，就能做出可以玩的五個樂齡互動裝置。大零件用鱷魚夾、小零件插麵包板，盡量不用焊接。

🚦 開始之前：每組都要先懂的事

1線的顏色代表什麼
紅線＝電（5V／正極＋） 黑線＝接地（GND／負極−） 綠線＝接燈／開關／訊號 黃線＝感測器/喇叭訊號
2先讓麵包板「有電」（很重要！）

每一組第一步都一樣：用紅線把 Arduino 的 5V 接到麵包板紅色電源軌、用黑線把 GND 接到藍色電源軌。麵包板上下兩組電源軌可以再用兩條短線互相接通，整塊板子就都有電了。之後所有零件要電、要接地，就近接這兩排。

紅線 5V → 紅排、黑線 GND → 藍排；上下電源軌各用一條短線接通。這是每組的共同第一步。
3小 LED 一定要配電阻

5mm 小 LED：長腳（＋）串一顆 220Ω 再接數位腳，短腳（−）接 GND。

4發光按鈕有 4 隻腳：開關 ＋ 燈，兩個都要接！

大按鈕底部有 4 隻腳：2 隻是「開關」（按下才接通，不分正負），2 隻是「燈」（有 ＋ 和 −）。每顆按鈕都要接這 4 條：開關接給 Arduino 偵測有沒有被按、燈接給 Arduino 控制亮滅。

一顆發光按鈕＝4 條線：開關 2 腳（一條到數位腳、一條到 GND）＋ 燈 2 腳（＋到數位腳、−到 GND）。燈在 5V 就會亮，直接接即可。
小提醒：想更保險，可在燈的 ＋腳串一顆 220Ω 再接腳位（會稍微暗一點）。所有「−／GND」都接到麵包板的藍色電源軌最整齊。
安全提醒：烙鐵很燙，焊接只由老師操作。學生負責夾鱷魚夾、插麵包板。接好線、確認紅黑沒有相碰後才插電。
1 不老神手 · 握力 PK

不老神手 — 一捏見高下 💪

林予晴 · 周艾蘿 · 吳依淳
🎯 玩法：按任一顆大按鈕開始，兩人同時用力捏壓力片約 5 秒，力氣大的人「按鈕」亮起 ＋ 叮咚。
📦材料清單
🗺️完整接線圖
線從到
紅Arduino 5V／兩片 FSR 各一腳麵包板紅排
黑Arduino GND／所有 −腳、FSR的10kΩ端麵包板藍排
黃FSR1 / FSR2 訊號A0 / A1
綠紅鈕燈＋ / 綠鈕燈＋D8 / D9
青紅鈕開關 / 綠鈕開關D10 / D11
黃蜂鳴器 ＋D7
🔌接線步驟
  1. 先讓麵包板有電：5V→紅排、GND→藍排（見「開始之前」）。
  2. 玩家1 FSR：一腳接紅排(5V)；另一腳接某排＋10kΩ到藍排(GND)，從交點拉黃線到 A0。
  3. 玩家2 FSR：同上，黃線到 A1。
  4. 紅鈕：燈＋→D8、開關→D10，燈−與開關另一腳都接藍排(GND)。
  5. 綠鈕：燈＋→D9、開關→D11，兩隻−腳接藍排(GND)。
  6. 蜂鳴器：＋→D7，−接藍排。檢查紅黑沒相碰→插電。
🎮怎麼玩
  • 按任一顆大按鈕，叮一聲開始（5 秒）。
  • 兩人同時用力捏壓力片。
  • 時間到，力氣大的那顆按鈕亮起＋叮咚。
  • 再按按鈕開始下一回合。
📄 老師用：程式碼（與 .ino 檔相同） 
// 第一組 不老神手 握力PK（按鈕：開關起跑、燈顯示贏家）
const int FSR_A=A0, FSR_B=A1;
const int LED_A=8, LED_B=9;     // 按鈕內建燈
const int BTN_A=10, BTN_B=11;   // 按鈕開關
const int BUZZER=7; const int TH=120;
bool playing=false; int maxA=0,maxB=0; unsigned long endT=0;
void setup(){
  pinMode(LED_A,OUTPUT);pinMode(LED_B,OUTPUT);
  pinMode(BTN_A,INPUT_PULLUP);pinMode(BTN_B,INPUT_PULLUP);
  pinMode(BUZZER,OUTPUT);Serial.begin(9600);
}
void loop(){
  if(!playing && (digitalRead(BTN_A)==LOW||digitalRead(BTN_B)==LOW)){
    playing=true;maxA=0;maxB=0;endT=millis()+5000UL;
    digitalWrite(LED_A,LOW);digitalWrite(LED_B,LOW);tone(BUZZER,1047,300);
  }
  if(playing){
    int a=analogRead(FSR_A),b=analogRead(FSR_B);
    if(a>maxA)maxA=a; if(b>maxB)maxB=b;
    if(millis()>endT){ playing=false;
      if(maxA>maxB&&maxA>TH){digitalWrite(LED_A,HIGH);tone(BUZZER,880,200);}
      else if(maxB>maxA&&maxB>TH){digitalWrite(LED_B,HIGH);tone(BUZZER,880,200);}
      else{digitalWrite(LED_A,HIGH);digitalWrite(LED_B,HIGH);tone(BUZZER,440,300);}
    }
  } delay(20);
}
2 輕輕鬆鬆 · 平衡感

平衡小球 — 手要穩、別歪掉 ⚖️

沈泓文 · 陳劭紘 · 楊浚玟
🎯 玩法：兩人各拿手把，手越平越穩；誰歪了發光小球往誰滾，滾到自己這端就輸。
這組用小 LED＋滾珠開關（沒有用到大按鈕）：5 顆小 LED 排成燈條當小球，滾珠開關偵測手有沒有歪。
📦材料清單
🗺️完整接線圖
線從到
紅 / 黑Arduino 5V / GND麵包板紅排 / 藍排
綠5 顆 LED（各串 220Ω，短腳→藍排）D3D4D5D6D7
綠滾珠A / 滾珠B（另一腳→藍排）D2 / D8
黃蜂鳴器 ＋D9
🔌接線步驟
  1. 先讓麵包板有電：5V→紅排、GND→藍排。
  2. 5 顆 LED 排一排，各串 220Ω 接 D3~D7（左→右），短腳都接藍排。D5 是中央起點。
  3. 滾珠A：一腳接藍排(GND)，一腳接 D2。
  4. 滾珠B：一腳接藍排(GND)，一腳接 D8。
  5. 蜂鳴器：＋→D9，−接藍排。
  6. 插電測試：放平不動、歪了小球往那邊滾；方向相反就把滾珠上下翻裝。
🎮怎麼玩
  • 手把放平，小球停中央。
  • 誰歪小球往誰滾一格（嗶）。
  • 撐穩撐正把小球推回去。
  • 滾到誰的末端誰就輸，全燈閃爍慶祝。
📄 老師用：程式碼 
// 第二組 平衡小球
const int TILT_A=2, TILT_B=8, BUZZER=9;
const int LED[5]={3,4,5,6,7};
int pos=2;
void show(){for(int i=0;i<5;i++)digitalWrite(LED[i],i==pos?HIGH:LOW);}
void win(){for(int k=0;k<6;k++){for(int i=0;i<5;i++)digitalWrite(LED[i],HIGH);tone(BUZZER,1047,150);delay(180);
  for(int i=0;i<5;i++)digitalWrite(LED[i],LOW);delay(120);}pos=2;show();}
void setup(){pinMode(TILT_A,INPUT_PULLUP);pinMode(TILT_B,INPUT_PULLUP);
  for(int i=0;i<5;i++)pinMode(LED[i],OUTPUT);pinMode(BUZZER,OUTPUT);show();}
void loop(){
  if(digitalRead(TILT_A)==HIGH&&pos>0){pos--;tone(BUZZER,500,60);}
  if(digitalRead(TILT_B)==HIGH&&pos<4){pos++;tone(BUZZER,500,60);}
  show(); if(pos==0||pos==4)win(); delay(150);
}

// 第三組 甩手棒 PK（按鈕：開關起跑、燈顯示贏家）
const int STICK_A=2, STICK_B=3;
const int LED_A=8, LED_B=9;     // 按鈕內建燈
const int BTN_A=10, BTN_B=11;   // 按鈕開關
const int BUZZER=7;
long cA=0,cB=0; int lA=HIGH,lB=HIGH; bool run=false; unsigned long endT=0;
void blinkWin(int led){for(int i=0;i<6;i++){digitalWrite(led,HIGH);tone(BUZZER,1047,150);delay(180);digitalWrite(led,LOW);delay(120);}}
void setup(){pinMode(STICK_A,INPUT_PULLUP);pinMode(STICK_B,INPUT_PULLUP);
  pinMode(BTN_A,INPUT_PULLUP);pinMode(BTN_B,INPUT_PULLUP);
  pinMode(LED_A,OUTPUT);pinMode(LED_B,OUTPUT);pinMode(BUZZER,OUTPUT);}
void loop(){
  if(!run && (digitalRead(BTN_A)==LOW||digitalRead(BTN_B)==LOW)){
    cA=0;cB=0;run=true;endT=millis()+60000UL;
    digitalWrite(LED_A,HIGH);digitalWrite(LED_B,HIGH);tone(BUZZER,1047,300);
    digitalWrite(LED_A,LOW);digitalWrite(LED_B,LOW);
  }
  if(run){
    int a=digitalRead(STICK_A),b=digitalRead(STICK_B);
    if(a!=lA){cA++;tone(BUZZER,1500,15);}lA=a;
    if(b!=lB){cB++;tone(BUZZER,1800,15);}lB=b;
    if(millis()>endT){run=false;
      if(cA>cB)blinkWin(LED_A);else if(cB>cA)blinkWin(LED_B);else tone(BUZZER,440,500);}
  } delay(5);
}
4 大家動一動 · 跟著拍燈

大家動一動 — 哪顆亮就拍哪顆 🙌

邱宇樂 · 鄭羽芹 · 鍾政穎
🎯 玩法：三顆大按鈕放在低/中/高（或左中右）。哪顆亮，長輩就伸手拍那顆，拍對叮咚、拍錯短嗶，再換下一顆亮。
原構想無法用 Arduino 做（Webcam＋AI）。改成「跟著燈拍按鈕」：把按鈕放在不同高度，長輩要伸展手臂去拍，運動到肩膀和手臂。每顆按鈕的開關和燈都會用到。

// 第四組 大家動一動（亮燈就拍）
const int LED[3]={5,6,7};    // 按鈕內建燈

const int BUZZER=11; int target=0;
void newTarget(){for(int i=0;i<3;i++)digitalWrite(LED[i],LOW);target=random(3);digitalWrite(LED[target],HIGH);}
void setup(){for(int i=0;i<3;i++){pinMode(LED[i],OUTPUT);pinMode(BTN[i],INPUT_PULLUP);}
  pinMode(BUZZER,OUTPUT);randomSeed(analogRead(A0));newTarget();}
void loop(){
  for(int i=0;i<3;i++){
    if(digitalRead(BTN[i])==LOW){

  }
}

// 第五組 夜市保衛戰 記憶遊戲
const int BTN[3]={2,3,4};
const int LED[3]={8,9,10};
const int BUZZER=7;
int seq[30],len=0; int notes[3]={523,659,784};
void flash(int i,int t){digitalWrite(LED[i],HIGH);tone(BUZZER,notes[i],t);delay(t);digitalWrite(LED[i],LOW);delay(160);}
void playSeq(){delay(600);for(int i=0;i<len;i++)flash(seq[i],400);}
void addStep(){seq[len++]=random(3);playSeq();}
void newGame(){len=0;addStep();}
int waitBtn(){while(true){for(int i=0;i<3;i++)if(digitalRead(BTN[i])==LOW){flash(i,200);while(digitalRead(BTN[i])==LOW);return i;}}}
void setup(){for(int i=0;i<3;i++){pinMode(BTN[i],INPUT_PULLUP);pinMode(LED[i],OUTPUT);}
  pinMode(BUZZER,OUTPUT);randomSeed(analogRead(A0));delay(500);newGame();}
void loop(){
  for(int i=0;i<len;i++){
    if(waitBtn()!=seq[i]){for(int k=0;k<3;k++){tone(BUZZER,200,250);delay(300);}delay(800);newGame();return;}
  }
  tone(BUZZER,1047,120);delay(150);tone(BUZZER,1319,200);delay(700);addStep();
}
