自己動手做：接線手冊 🛠️

照著圖一條線一條線接，就能做出可以玩的五個樂齡互動裝置。大零件用鱷魚夾、小零件插麵包板，盡量不用焊接。

🧲 大零件用鱷魚夾🍞 小零件插麵包板🌈 接線照顏色對💡 發光按鈕會亮
先看這裡 第一組第二組第三組第四組第五組

🚦 開始之前：每組都要先懂的事

1線的顏色代表什麼
紅線＝電（5V／正極＋） 黑線＝接地（GND／負極−） 綠線＝接燈／開關／訊號 黃線＝感測器/喇叭訊號
2先讓麵包板「有電」（很重要！）

每一組第一步都一樣：用黑線把 Arduino 的 GND 接到麵包板「下排藍(−)」；需要電的組別（第一組）再用紅線把 5V 接到「下排紅(＋)」，並用短線繞過左邊把上排接通。接線圖裡的「e4」代表第 4 直排的 e 孔——同一直排的 a~e 五個孔互相相通（f~j 也相通），照圖把線插進一樣的孔就對了。

黑線 GND→下排藍(−)、紅線 5V→下排紅(＋)（從 Arduino 右側繞過去），左邊兩條短線把上排紅藍也接通。圖上的 a~j、1~30 跟真的麵包板一樣。
⚠️ 你們的麵包板（60排長板）中間有缺口：紅藍電源軌在板子正中央是「斷開」的（看得到紅藍線中間有空隙）。開工第一件事：在四條電源軌的缺口上各跨一條短線（至少把兩條藍(−)接通）。接線圖畫的是板子的左半邊 1~30 排——把板子轉成小號碼在左手邊，圖上的排數、a~j 字母就跟板子一模一樣。
3小 LED 一定要配電阻

5mm 小 LED：長腳（＋）串一顆 220Ω 再接數位腳，短腳（−）接 GND。

4發光按鈕有 4 隻腳：開關 ＋ 燈，兩個都要接！

大按鈕底部有 4 個接點：2 個是「開關」（插片上標 COM 和 NO，按下才接通，兩片不分順序），2 個是「燈」（LED 的 ＋ 和 −）。每顆按鈕都要接這 4 條：開關接給 Arduino 偵測有沒有被按、燈接給 Arduino 控制亮滅。

一顆發光按鈕＝4 條線：開關 COM、NO 兩插片（一條到數位腳、一條到 GND，順序可互換）＋ 燈 2 腳（＋到數位腳、−到 GND）。
小提醒：所有「−／GND」都接到麵包板的藍色電源軌最整齊。若燈接上 5V 很暗或不亮，可能是 12V 規格的燈芯——遊戲一樣能玩，只是那顆不亮；之後可換成 5V 燈芯。
安全提醒：烙鐵很燙，焊接只由老師操作。學生負責夾鱷魚夾、插麵包板。接好線、確認紅黑沒有相碰後才插電。
1 不老神手 · 握力 PK

不老神手 — 一捏見高下 💪

林予晴 · 周艾蘿 · 吳依淳
🎯 玩法：按任一顆大按鈕開始，兩人同時用力捏壓力片約 5 秒，力氣大的人「按鈕」亮起 ＋ 叮咚。
📦材料清單
🗺️完整接線圖
孔位代號：e4＝第4直排的e孔（同一直排 a~e 相通）。FSR 在第4、9直排；蜂鳴器在第22、24直排。
線從到
紅Arduino 5V／兩片 FSR 各一腳麵包板紅排
黑Arduino GND／所有 −腳、FSR的10kΩ端麵包板藍排
黃FSR1 / FSR2 訊號A0 / A1
綠紅鈕燈＋ / 綠鈕燈＋D8 / D9
青紅鈕開關 / 綠鈕開關D10 / D11

🔌接線步驟
  1. 電源：紅線 5V→下排紅(＋)、黑線 GND→下排藍(−)（從 Arduino 右邊繞過去）；再用一條黑短線繞左邊，把「上排藍(−)」和「下排藍(−)」接通。先用短線跨過電源軌中間的缺口（見「開始之前」）。
  2. 玩家1 FSR：一腳→下排紅(＋)；另一腳→e4。10kΩ 電阻一端插 b4、另一端插下排藍(−)。黃線從 a4→A0。
  3. 玩家2 FSR：一腳→下排紅(＋)；另一腳→e9。10kΩ：b9→下排藍(−)。黃線 a9→A1。
  4. 紅鈕(玩家1)：燈＋→D8、開關→D10；燈−和開關另一腳都插「上排藍(−)」。
  5. 綠鈕(玩家2)：燈＋→D9、開關→D11；兩隻−腳插上排藍(−)。
  6. 蜂鳴器：＋腳插 b22、−腳插 b24；黃線 a22→D7、黑線 a24→下排藍(−)。檢查紅黑沒相碰→插電。
🎮怎麼玩
  • 按任一顆大按鈕，叮一聲開始（5 秒）。
  • 兩人同時用力捏壓力片。
  • 時間到，力氣大的那顆按鈕亮起＋叮咚。
  • 再按按鈕開始下一回合。

// 第一組 不老神手 握力PK（按鈕：開關起跑、燈顯示贏家）
const int FSR_A=A0, FSR_B=A1;
const int LED_A=8, LED_B=9;     // 按鈕內建燈
const int BTN_A=10, BTN_B=11;   // 按鈕開關
const int BUZZER=7; const int TH=120;
bool playing=false; int maxA=0,maxB=0; unsigned long endT=0;
void setup(){
  pinMode(LED_A,OUTPUT);pinMode(LED_B,OUTPUT);
  pinMode(BTN_A,INPUT_PULLUP);pinMode(BTN_B,INPUT_PULLUP);
  pinMode(BUZZER,OUTPUT);Serial.begin(9600);
}
void loop(){
  if(!playing && (digitalRead(BTN_A)==LOW||digitalRead(BTN_B)==LOW)){
    playing=true;maxA=0;maxB=0;endT=millis()+5000UL;
    digitalWrite(LED_A,LOW);digitalWrite(LED_B,LOW);tone(BUZZER,1047,300);
  }
  if(playing){
    int a=analogRead(FSR_A),b=analogRead(FSR_B);
    if(a>maxA)maxA=a; if(b>maxB)maxB=b;
    if(millis()>endT){ playing=false;
      if(maxA>maxB&&maxA>TH){digitalWrite(LED_A,HIGH);tone(BUZZER,880,200);}
      else if(maxB>maxA&&maxB>TH){digitalWrite(LED_B,HIGH);tone(BUZZER,880,200);}
      else{digitalWrite(LED_A,HIGH);digitalWrite(LED_B,HIGH);tone(BUZZER,440,300);}
    }
  } delay(20);
}
2 輕輕鬆鬆 · 平衡感

平衡小球 — 手要穩、別歪掉 ⚖️

沈泓文 · 陳劭紘 · 楊浚玟
🎯 玩法：兩人各拿手把，手越平越穩；誰歪了發光小球往誰滾，滾到自己這端就輸。
這組用小 LED＋滾珠開關（沒有用到大按鈕）：5 顆小 LED 排成燈條當小球，滾珠開關偵測手有沒有歪。
📦材料清單
🗺️完整接線圖
每顆 LED 一組四件：腳位綠線(a孔)→220Ω(b排)→LED長腳(c孔)，短腳旁排黑線回下排藍(−)。最左邊那顆接 7 號腳。
線從到
紅 / 黑Arduino 5V / GND麵包板紅排 / 藍排
綠5 顆 LED（左→右，各串 220Ω）D7D6D5D4D3
綠滾珠模組A / B 的 DO（VCC→紅排、GND→藍排）D2 / D8
黃蜂鳴器 ＋D9
🔌接線步驟
  1. 電源：紅線 5V→下排紅(＋)（滾珠模組要用）、黑線 GND→下排藍(−)（從 Arduino 右邊繞過去）。先用短線跨過電源軌中間的缺口（見「開始之前」）。
  2. LED×5（左→右接 D7、D6、D5、D4、D3）：每顆插法相同——腳位綠線插 a孔、220Ω 跨在 b排、LED 長腳(＋)插 c孔、短腳(−)旁邊那排用黑短線回下排藍(−)。照圖：7號腳→a3、電阻 b3→b5、LED 長腳 c5 短腳 c6、黑線 a6→藍(−)；其餘四顆在 a9、a15、a21、a27 照同樣間隔插。中間（接 5 號腳）那顆是起點。
  3. 滾珠模組A：VCC→下排紅(＋)、GND→下排藍(−)、DO→D2。（板上的藍色小旋鈕不用轉）
  4. 滾珠模組B：VCC→下排紅(＋)、GND→下排藍(−)、DO→D8。
  5. 蜂鳴器：＋腳插 c1、−腳插 c2；黃線 a1→D9、黑線 a2→下排藍(−)。
  6. 插電測試：放平不動、歪了小球往那邊滾；方向相反就把滾珠上下翻裝。
🎮怎麼玩
  • 手把放平，小球停中央。
  • 誰歪小球往誰滾一格（嗶）。
  • 撐穩撐正把小球推回去。
  • 滾到誰的末端誰就輸，全燈閃爍慶祝。

const int TILT_A=2, TILT_B=8, BUZZER=9;

void win(){for(int k=0;k<6;k++){for(int i=0;i<5;i++)digitalWrite(LED[i],HIGH);tone(BUZZER,1047,150);delay(180);
  for(int i=0;i<5;i++)digitalWrite(LED[i],LOW);delay(120);}pos=2;show();}

3 樂齡 · 甩手棒 PK

甩手棒 — 一分鐘甩越多越棒 🏏

黃宇昕 · 饒語芯 · 陳之謙
🎯 玩法：按任一顆大按鈕開始，LCD 螢幕顯示 START! 後倒數 60 秒並即時顯示兩人甩的次數；時間到，甩較多次者的「按鈕」閃爍、螢幕顯示贏家。
老師先做一次：這組程式用到 LCD 程式庫——Arduino IDE 開「工具 → 管理程式庫」，搜尋「LiquidCrystal I2C」（Frank de Brabander 版）按安裝。LCD 沒畫面：把程式裡的 0x27 改成 0x3F；有亮但沒字：轉 LCD 背面的對比旋鈕。
📦材料清單
🗺️完整接線圖
按鈕的−腳都插「上排藍(−)」，左邊短線把上下藍(−)接通。蜂鳴器＋插b24、−插b26。LCD 四條線：GND→GND腳、VCC→下排紅(＋)、SDA→A4、SCL→A5。
線從到
紅 / 黑Arduino 5V / GND（含模組 VCC、所有 −腳）下排紅(＋) / 藍(−)
綠甩手棒A / B 模組的 DO（VCC→紅排、GND→藍排）D2 / D3
綠紅鈕燈＋ / 綠鈕燈＋D8 / D9

  1. 電源：紅線 5V→下排紅(＋)（滾珠模組要用）、黑線 GND→下排藍(−)，再用黑短線繞左邊接通「上排藍(−)」。先用短線跨過電源軌中間的缺口。
  2. LCD 螢幕（I2C）：GND→Arduino 的 GND 腳（5V 旁邊那兩個都可以）、VCC→下排紅(＋)、SDA→A4、SCL→A5。四條都用杜邦母頭直接插。
  3. 甩手棒A、B（滾珠模組）：VCC→下排紅(＋)、GND→下排藍(−)、DO 各→D2、D3。（藍色小旋鈕不用轉）

  5. 綠鈕(玩家B)：燈＋→D9、開關→D11，兩隻−腳插上排藍(−)。
  6. 蜂鳴器：＋腳插 b24、−腳插 b26；黃線 a24→D7、黑線 a26→下排藍(−)。
🎮怎麼玩
  • 按任一顆大按鈕 → 螢幕顯示「START!」，開始 60 秒。
  • 用力甩棒，每甩一下嗶一聲；螢幕同時顯示「TIME: 剩餘秒數」和「A:次數 B:次數」。
  • 時間到，甩較多次者的按鈕閃爍＋勝利音效，螢幕顯示「A WINS!」或「B WINS!」（平手 TIE!）。
  • 稍等螢幕回到 PRESS BUTTON!，再按按鈕開始下一場。

// 第三組 甩手棒 PK（LCD 顯示 START! 與倒數）
// ★先安裝程式庫：管理程式庫→搜尋 LiquidCrystal I2C
#include <Wire.h>
#include <LiquidCrystal_I2C.h>
LiquidCrystal_I2C lcd(0x27,16,2);  // 沒畫面就改 0x3F

const int STICK_A=2, STICK_B=3;
const int LED_A=8, LED_B=9;     // 按鈕內建燈
const int BTN_A=10, BTN_B=11;   // 按鈕開關
const int BUZZER=7;
const unsigned long GAME_MS=60000UL;  // 60秒（可調）
long countA=0,countB=0; int lastA=HIGH,lastB=HIGH;
bool running=false; unsigned long endTime=0; int lastSec=-1;

void showReady(){ lcd.clear();
  lcd.setCursor(0,0); lcd.print("SWING  PK");
  lcd.setCursor(0,1); lcd.print("PRESS BUTTON!"); }
void showCounts(){ lcd.setCursor(0,1);
  lcd.print("A:");lcd.print(countA);lcd.print("  B:");lcd.print(countB);lcd.print("   "); }
void finish(){ running=false; lcd.clear();
  lcd.setCursor(0,1); lcd.print("A:");lcd.print(countA);lcd.print("  B:");lcd.print(countB);
  lcd.setCursor(0,0); int led=-1;
  if(countA>countB){lcd.print("A WINS!");led=LED_A;}
  else if(countB>countA){lcd.print("B WINS!");led=LED_B;}
  else lcd.print("TIE!");
  if(led>=0){for(int i=0;i<6;i++){digitalWrite(led,HIGH);tone(BUZZER,1047,150);delay(180);
    digitalWrite(led,LOW);delay(120);}}
  else tone(BUZZER,440,500);
  delay(1500); showReady(); }

void setup(){
  pinMode(STICK_A,INPUT_PULLUP);pinMode(STICK_B,INPUT_PULLUP);
  pinMode(BTN_A,INPUT_PULLUP);pinMode(BTN_B,INPUT_PULLUP);
  pinMode(LED_A,OUTPUT);pinMode(LED_B,OUTPUT);pinMode(BUZZER,OUTPUT);
  lcd.init();  // 部分版本要改 lcd.begin();
  lcd.backlight(); showReady();
}
void loop(){
  if(!running && (digitalRead(BTN_A)==LOW||digitalRead(BTN_B)==LOW)){
    countA=0;countB=0;lastSec=-1;
    lcd.clear();lcd.setCursor(5,0);lcd.print("START!");
    digitalWrite(LED_A,HIGH);digitalWrite(LED_B,HIGH);
    tone(BUZZER,1047,300);delay(800);
    digitalWrite(LED_A,LOW);digitalWrite(LED_B,LOW);
    lcd.clear();showCounts();
    running=true; endTime=millis()+GAME_MS;
  }
  if(running){
    int a=digitalRead(STICK_A),b=digitalRead(STICK_B);
    bool changed=false;
    if(a!=lastA){countA++;tone(BUZZER,1500,15);changed=true;}lastA=a;
    if(b!=lastB){countB++;tone(BUZZER,1800,15);changed=true;}lastB=b;
    long remain=(long)(endTime-millis()); if(remain<0)remain=0;
    int sec=(remain+999)/1000;
    if(sec!=lastSec){ lastSec=sec;          // 每秒更新倒數
      lcd.setCursor(0,0);lcd.print("TIME: ");lcd.print(sec);lcd.print("s   "); }
    if(changed)showCounts();              // 次數有變才更新
    if(remain==0)finish();
  }
  delay(5);
}

5 夜市保衛戰 · 搶答機

夜市保衛戰 — 二人搶答機 ⚡

王品喬 · 邱湘甯 · 周莉娜
🎯 玩法：主持人唸出「紙本任務卡」的題目，兩人搶按大按鈕——先按到的人按鈕亮起＋音效（鎖定），就由他回答；約 10 秒後自動重置換下一題。
每顆發光按鈕都用到 4 隻腳：「開關」給 Arduino 判斷誰先按到，「燈」由程式點亮先搶到的那顆。任務卡用紙卡自己做：正面畫圖、背面寫題目，跟夜市主題搭配（例：這攤賣什麼？台語怎麼說？）。
📦材料清單
🗺️完整接線圖
孔位代號：b24＝第24直排的b孔（同一直排 a~e 相通）。按鈕的−腳都插「上排藍(−)」，左邊短線把上下藍(−)接通。
線從到
黑Arduino GND／按鈕所有−腳下排藍(−)／上排藍(−)
青A鈕開關 / B鈕開關D2 / D3
綠A鈕燈＋ / B鈕燈＋D8 / D9
黃蜂鳴器＋（腳插b24，線從a24）D7
🔌接線步驟
  1. 電源：黑線 GND→下排藍(−)（從 Arduino 右邊繞過去），再用黑短線繞左邊接通「上排藍(−)」。這組不用 5V。先用短線跨過電源軌中間的缺口。
  2. 玩家A 紅鈕：開關→D2、燈＋→D8；兩隻−腳插上排藍(−)（圖中第 2、4 直排）。
  3. 玩家B 綠鈕：開關→D3、燈＋→D9；兩隻−腳插上排藍(−)（第 27、29 直排）。
  4. 蜂鳴器：＋腳插 b24、−腳插 b26；黃線 a24→D7、黑線 a26→下排藍(−)。
  5. 插電測試：兩顆鈕快閃兩下＝「可以搶了！」。
🎮怎麼玩
  • 主持人抽一張任務卡，唸出題目。
  • 喊「開始」後兩人搶按——先按到的人按鈕亮起＋「叮叮」，另一人再按也沒用（已鎖定）。
  • 亮燈的人在 10 秒內回答；答對就把卡片拿走當分數。
  • 聽到「叮咚」＋兩鈕快閃兩下＝重新開放搶答，換下一題。

小撇步：程式裡的 ANSWER_MS 可以調整回答秒數（10000＝10秒）。

📄 老師用：程式碼（與 .ino 檔相同） 
// 第五組 夜市保衛戰 二人搶答機
const int BTN_A=2, BTN_B=3;     // 按鈕開關
const int LED_A=8, LED_B=9;     // 按鈕內建燈
const int BUZZER=7;
const unsigned long ANSWER_MS=10000UL;  // 回答時間10秒（可調）

void readySignal(){            // 「可以搶了！」兩燈快閃兩下
  for(int i=0;i<2;i++){
    digitalWrite(LED_A,HIGH);digitalWrite(LED_B,HIGH);
    tone(BUZZER,784,80);delay(130);
    digitalWrite(LED_A,LOW);digitalWrite(LED_B,LOW);delay(130);
  }
}
void winner(int led,int freq){   // 有人搶到了
  digitalWrite(led,HIGH);          // 只亮贏家的鈕（鎖定）
  tone(BUZZER,freq,250);delay(300);tone(BUZZER,freq,250);
  delay(ANSWER_MS);                // 回答任務卡題目的時間
  digitalWrite(led,LOW);
  tone(BUZZER,523,120);delay(160);tone(BUZZER,659,120);delay(400);
  readySignal();                   // 換下一題
}
void setup(){
  pinMode(BTN_A,INPUT_PULLUP);pinMode(BTN_B,INPUT_PULLUP);
  pinMode(LED_A,OUTPUT);pinMode(LED_B,OUTPUT);pinMode(BUZZER,OUTPUT);
  readySignal();
}
void loop(){
  if(digitalRead(BTN_A)==LOW)      winner(LED_A,880);  // A 先按到
  else if(digitalRead(BTN_B)==LOW) winner(LED_B,988);  // B 先按到
}
